Class: DxlRequestNode

DxlRequestNode(nodeConfig)

Node which sends a request message containing the msg.payload from the input message to a remote DXL service and writes the response received from the request to the output msg.payload property.

Constructor

new DxlRequestNode(nodeConfig)

Parameters:
Name Type Description
nodeConfig Object

Configuration data which the node uses.

Properties
Name Type Attributes Default Description
returnType String <optional>
 txt

Controls the data type for the msg.payload property in the new message injected into a flow. If returnType is 'bin', the raw binary Buffer received in the DXL response payload is forwarded along. If returnType is 'txt', the binary Buffer is decoded from UTF-8 octets into a String. If returnType is 'obj', the binary Buffer is decoded into a UTF-8 string and parsed as JSON text into an Object. If an error occurs when attempting to convert the binary Buffer of the payload into the desired data type, the current flow is halted with an error.
topic String

Topic to send the request to. If the value is empty, the topic will be derived from the input message's msg.dxlTopic property.
client String

Id of the DXL client configuration node that this node should be associated with.
